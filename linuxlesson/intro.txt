GIT - распределенная система контроля версий 
это система для отлеживания и ведения истории изменения файлов, в вашем проекте.
Репозиторий - хранилище вашего кода и истории его измений
GitHub - веб сайт для размещения git-репозиториев в совместной разработке проектов. 
Команды Git:
 1. git init - она создает новый локальный гит репозиторий у вас на ПК, в той директорий в которой была прописана команда. (Команда вводится один раз а начале).
 2. git add - команда для добавления файлов в систему отслеживания гита после этой команды изменения начинают отслеживаться гитом 
 3. git commit -m '<comment>'как только достигаем определенного момента в разработке, то тогда мы сохраняем и комментируем изменения в локальный репозиторий 
 4. git remote add - это команда для того чтобы связать наш локальеый репозиторий с удаленным репозиторием, то есть добавляет подключение к репозиторий на гитхабе. git remote add <name: origin> <ссылка_на_репо:ssh>
 5. git push - команда для отправки всех изменений на локальеом репозитории в удаленный репозиторий  git push origin main
