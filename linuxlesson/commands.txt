 Linux commands 
 Ctl + alt +T - команда открытия терминала команда открытия терминала
 ~ - символ домашней директории 
 / - символ корневой директории
 .. - обозначают предыдущие директории 
 . - обозначает текущую директорию
 1. ls(list) - команда для вывода всех директорий (папка) и файлов в текущей (рабочей) директории в терминале 
 2. cd (change director) - комнда для изменения (переходов) одной директории на другую cd <название директории>
 3. mkdir - (make diretory) - создает указанную директорию в рабочей директории в терминале 
 4. touch - создает указанный файл 
 5. rm - удаление файла
 6. rmdir - удаление директории 
 7. pwd (present working dirextory) - выводит путь до текущей директории 
 8. rm-rf -  принудительное удаление 
 9. ls -a - вывидодит список как скрытых так и обычных файлов и директорий 
 10. nano <path_to_file> - откроется файл для редактирования в графическом интерфейсе 
 * если нет файла то nano попытается создать новый файл 
 11. cat <file_name> - вфводит содержимое файла в терминале 
 12. echo - это команда для быстрой записи текстовой информации в какой либо файл. 	echo текст_для_записи > название_файла  
 13. mv - команда для переименовывания файла либо директории, а так же для перемещения файлов или директории. переименовывание: mv jonh john - mv<старое_название> <новое_название> Перемещение: mv john ~/Desktop - mv <file/directory> <directory>
